# 🌍 NGO-Website-Bootstrap

A modern, fully responsive NGO & Charity website built using **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**. This project is designed to help nonprofit organizations present their mission, showcase donation campaigns, promote volunteer opportunities, and connect with supporters through a clean and user-friendly interface.
